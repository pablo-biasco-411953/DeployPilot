# Configuracion del server

El server tendra setup rapido y setup avanzado.

## Setup rapido

1. Elegir InMemory, MySQL o Postgres.
2. Ingresar connection string.
3. Elegir carpeta de artefactos.
4. Crear usuario admin.
5. Registrar primer repositorio Git.
6. Elegir plantilla de build.

## Proveedores de persistencia

- `InMemory`: ideal para demo local y tests.
- `Postgres`: recomendado para produccion y Supabase.
- `MySql`: util para equipos que ya usan infraestructura MySQL o compatible.
